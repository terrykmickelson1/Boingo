Privacy Policy

Application: Boingo Finance
Last updated: April 2026

1. Overview

Boingo Finance is a private, personal-use application with no end users other than the registered members of the Boingo household. It is not a commercial product.

2. Information We Access

This Application accesses financial account balances entered directly by the user and retrieved via the QuickBooks Online API for the user's own company file. No personally identifiable information is collected from any third party.

3. Data Storage

All data is stored locally in a SQLite database on the user's own device or private Streamlit Cloud instance. Data is not stored on shared servers or transmitted to third parties except:

  • Intuit QuickBooks Online — via their official OAuth 2.0 API, solely to read and write journal entries for the authenticated user's own company file.
  • Streamlit Community Cloud — solely for hosting the application interface for the authenticated user.

4. Data Sharing

No financial data, personal information, or usage data is sold, rented, shared, or disclosed to any third party for any purpose.

5. Security

OAuth 2.0 tokens are stored in a private, gitignored configuration file and are never logged or transmitted beyond the authorized API endpoints.

6. Your Rights

As the sole user of this Application, you retain full ownership and control of all data. You may delete all stored data at any time by removing the local database file.

7. Contact

This application is privately operated. For questions, contact the application owner directly.
