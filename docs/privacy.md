# Privacy Policy — Boingo Finance

**Last updated: April 2026**

## Overview

Boingo Finance is a private, personal-use application. It is not a commercial product and has no end users other than the registered household members of the Boingo family unit.

## Data Collected

This application accesses and displays financial account balances provided by the user and retrieved via the QuickBooks Online API. No personally identifiable information is collected from any third party.

## Data Storage

All data is stored locally in a SQLite database on the user's own device or private Streamlit Cloud instance. No data is stored on shared servers or transmitted to third parties other than:

- **Intuit QuickBooks Online** — via their official OAuth 2.0 API, solely to read and write journal entries for the authenticated user's own company file.
- **Streamlit Community Cloud** — solely for hosting the application interface for the authenticated user.

## Data Sharing

No financial data, personal information, or usage data is sold, rented, or shared with any third party for any purpose.

## Security

OAuth 2.0 tokens are stored in a private configuration file and are never logged or transmitted beyond the authorized API endpoints.

## Contact

This application is privately operated. For questions, contact the application owner directly.
